# FaceAntiSpoofingModel

### Requirements

- Python 3.6+
- OpenCV
- Numpy
- PyTorch

### Training the Model
1. Run `python Train.py`
2. After Training is complete the program will generate the file "./MyModel.pth", containing weights of the model

### Recognizing
1. Run `python Test.py`
